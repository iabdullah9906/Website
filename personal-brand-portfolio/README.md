# Personal Brand Portfolio Website

This is a multi-page HTML and CSS website built to feel personal and versatile.

## Pages
- `index.html`
- `about.html`
- `projects.html`
- `gallery.html`
- `interests.html`
- `contact.html`

## CSS
- `assets/css/style.css`

## Image files used
Add your own images to `assets/images/` with these names:

- `profile.jpg`
- `about-main.jpg`
- `project-web.jpg`
- `project-java.jpg`
- `project-car.jpg`
- `gallery-1.jpg`
- `gallery-2.jpg`
- `gallery-3.jpg`
- `gallery-4.jpg`
- `gallery-5.jpg`
- `gallery-6.jpg`

## Before uploading to GitHub
Change:
- your email
- your LinkedIn URL
- your GitHub URL
- your project links
- your project descriptions
- any text you want to personalize more

## Why this version is better
This version is not just a school portfolio. It gives you room to show:
- your technical work
- your soccer side
- your car projects
- your personal brand
- your images and screenshots

## How to use
1. Download the zip
2. Open the folder in VS Code
3. Add your images into `assets/images`
4. Update the text and links
5. Upload to GitHub
6. Deploy with Netlify or Vercel
